# Security Event and Incident Analysis

## Project Overview

This project analyzes cybersecurity events and incidents to identify indicators of compromise (IoCs), investigate suspicious activity, and recommend appropriate response actions.

## Objectives

- Analyze security events
- Investigate potential incidents
- Identify Indicators of Compromise (IoCs)
- Recommend containment and remediation strategies

## Skills Demonstrated

- Security Monitoring
- Incident Response
- Log Analysis
- Threat Detection
- Digital Forensics Fundamentals
- Cybersecurity Documentation

## Technologies & Frameworks

- SIEM Concepts
- NIST Incident Response Framework
- MITRE ATT&CK
- Security Event Analysis

## Key Deliverables

- Incident Analysis Report
- Security Findings
- Response Recommendations
- Lessons Learned
