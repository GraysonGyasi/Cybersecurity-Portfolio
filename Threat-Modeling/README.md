# Threat Modeling

## Project Overview

Performed a threat modeling assessment of the TRex Smart Everything (SE) SaaS platform hosted in Microsoft Azure.

## Objectives

- Identify security threats
- Analyze attack surfaces
- Evaluate risks
- Recommend mitigations

## Skills Demonstrated

- Threat Modeling
- STRIDE
- Risk Assessment
- Cloud Security
- Security Architecture

## Tools & Technologies

- Microsoft Threat Modeling Tool
- Azure Architecture
- STRIDE
- NIST
- Risk Analysis

## Key Deliverables

- Threat Modeling Report
- Threat Matrix
- Risk Assessment
- Security Recommendations

## Outcome

Identified multiple attack vectors and recommended mitigation strategies to improve the confidentiality, integrity, and availability of the cloud environment.
